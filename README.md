FITUR WEBSITE:
Website digunakan untuk registrasi prom night dan validasi tiket digital.

FLOW USER:
1. User membuka landing page
2. Melihat detail acara
3. Klik tombol “Daftar Sekarang”
4. Mengisi form registrasi:
   - Jumlah tiket
   - Nama lengkap
   - Email
   - Nomor WhatsApp
5. Sistem menghitung total pembayaran otomatis:
   - 1 tiket = 85.000
6. User upload bukti pembayaran
7. Status:
   - Menunggu Verifikasi
8. Admin memverifikasi pembayaran
9. Jika valid:
   - Admin klik tombol “Kirim Barcode”
10. Sistem otomatis mengirim barcode/QR ticket ke Gmail user
11. Jika user membeli lebih dari 1 tiket:
   - Generate barcode berbeda untuk tiap tiket
   - Setiap barcode memiliki nama ticket holder berbeda
   - Namun tetap dikirim ke 1 email pemesan
12. Saat barcode di scan:
   - Tampilkan:
     “Pembayaran Berhasil”
     Nama pemilik tiket
     Status hadir
13. Jika barcode sudah pernah dipakai:
   - Tampilkan:
     “Barcode sudah digunakan”
